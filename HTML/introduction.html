HTML Introduction

HTML is the standard markup language for creating Web pages

What is HTML ?

  • HTML stands for Hyper Text Markup Language
  • HTML is standard markup language for creating web pages
  • HTML describes the structure pof a web pages
  • HTML consists of a series of element
  • HTML element tell the browser how to display content
  • HTML element label piecces of content susch as "this is a heading, "this is a paragraph", "this is a link", etc.

A Simple HTML Document

<html>
<head>
<title>Page Title</title>
<head>
<body>

<h1>My first Heading </h1>
<p> My first Paragraph </p>

</body>
</html>
